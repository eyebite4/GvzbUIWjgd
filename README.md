# 项目简介

## 核心功能
- 登录注册
- 学生管理
- 课程管理
- 学生成绩管理
- 选课信息管理
- 课程类型管理
- 公告管理
- 教师管理
- 个人中心

## 适用场景
- 教育机构
- 学校
- 教育信息化建设

## 技术特点
- 使用Django框架进行后端开发
- 采用MySQL作为数据库
- 前端采用Vue.js技术
- 代码可运行，需自行配置环境
